# 📚 스파르타 북큐레이션 서비스 기획서

> **"지금 내 고민, 팀스파르타 문화로 풀어줄 책 한 권"**
> 연차와 고민을 입력하면 팀스파르타의 핵심가치와 그라운드룰을 기반으로 AI가 딱 맞는 책을 추천해주는 사내 서비스

---

## 1. 서비스 개요

| 항목 | 내용 |
|------|------|
| **서비스명** | 스파르타 북큐레이션 |
| **목적** | 팀원의 성장 고민을 팀스파르타 문화(핵심가치 + 그라운드룰)와 연결해 맞춤 도서 추천 |
| **대상** | 팀스파르타 내부 구성원 전체 |
| **접근 방식** | 로그인 없음, URL 공유만으로 접근 가능 |
| **런칭 형태** | 웹 서비스 (모바일 반응형) |

---

## 2. 문제 정의

팀스파르타 구성원은 빠른 성장과 실행을 지향하지만, 각자의 연차와 상황에 따라 다양한 고민을 가진다.

- 신입/주니어: "어떻게 빠르게 성장하지? 실패가 두렵다."
- 미드레벨: "리더십이 뭔지 모르겠다. 내 역할이 뭔가?"
- 시니어/리더: "팀 문화를 어떻게 만들지? 피드백을 잘 하고 싶다."

**기존 해결 방식의 한계:** 책 추천이 개인 경험에 의존, 회사 문화와 연결되지 않은 일반적인 도서 추천

**솔루션:** 팀스파르타의 고유한 핵심가치와 그라운드룰을 AI에 학습시켜, 문화에 맞는 책을 추천

---

## 3. 핵심 기능

### 3-1. 메인 입력 화면
- **연차 선택**: 0~1년차 / 2~3년차 / 4~7년차 / 8년차 이상
- **고민 입력**: 자유 텍스트 입력 (최대 300자)
- **추천받기 버튼**: 클릭 시 AI 분석 시작

### 3-2. AI 추천 엔진 (Claude API)
AI가 다음 요소를 종합해 책을 추천:

1. **입력 데이터**: 연차 + 고민 내용
2. **팀스파르타 핵심가치**: 빠르게 / 와우하게 / 진정성있게
3. **인재상 5가지**: 말을 예쁘게 하는 사람 / 성장을 갈망하는 사람 / 실패에서 배우는 사람 / 유쾌한 사람 / 만들고 싶은 세상이 있는 사람
4. **그라운드룰 10가지**: 슬랙보다 대화 / 대안 제시 / R&R 유연성 / ETA 준수 / 피드백 자작 등

### 3-3. 추천 결과 화면
- **추천 도서 3권** (제목, 저자, 표지 이미지)
- **추천 이유**: "당신의 고민과 팀스파르타 그라운드룰 #8(피드백 자작)을 연결해..."
- **연결된 핵심가치/그라운드룰 태그** 표시
- **결과 공유하기** 버튼 (URL 복사 / 카카오톡 공유)
- **다시 추천받기** 버튼

### 3-4. 히스토리 저장 (Supabase)
- 입력한 고민 + 추천 결과를 익명으로 저장
- 추후 인기 고민 키워드 / 많이 추천된 책 통계 확인 가능

## 4. 사용자 플로우

```
[랜딩 페이지]
    ↓ 서비스 소개 + "시작하기" 버튼
[입력 화면]
    ↓ 연차 선택 + 고민 텍스트 입력 + "추천받기"
[로딩 화면]
    ↓ Claude AI 분석 중 (스파르타 감성 로딩 애니메이션)
[결과 화면]
    ↓ 책 3권 + 추천 이유 + 연결 키워드
[공유/재시작]
    → URL 공유 or 다시 시도
```

| 영역 | 기술 | 역할 |

## 6. 데이터 구조 (Supabase)

### recommendations 테이블
```sql
CREATE TABLE recommendations (
  id          UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  years_exp   VARCHAR(20) NOT NULL,   -- "0~1년차", "2~3년차" 등
  concern     TEXT NOT NULL,           -- 사용자 고민 텍스트
  books       JSONB NOT NULL,          -- AI 추천 결과 (책 3권 JSON)
  core_values TEXT[],                  -- 연결된 핵심가치 태그
  ground_rules TEXT[],                 -- 연결된 그라운드룰 태그
  created_at  TIMESTAMP DEFAULT NOW()
);
```

## 7. Claude AI 프롬프트 전략

### 시스템 프롬프트 구조
```
당신은 팀스파르타의 도서 큐레이터입니다.

[팀스파르타 핵심가치]
- 빠르게: 아이디어를 작고 빠르게 검증, 간결한 실행
- 와우하게: 고객이 예측하지 못한 순간의 감동
- 진정성있게: 고객보다 고객을 더 생각하는 자세

[인재상 - 함께 밥 먹고 싶은 사람]
1. 말을 예쁘게 하는 사람
2. 성장을 갈망하는 사람
3. 실패에서 배우는 사람
4. 유쾌한 사람
5. 만들고 싶은 세상이 있는 사람

[그라운드룰]
1. 슬랙 100번보다 잡담 1번이 낫다
2. 안돼요 대신 "이렇게 하면 어때요?"
3. 네 일 내 일 하다 내일 된다
... (10가지 전체)

위 문화를 기반으로, 아래 입력에 맞는 책 3권을 추천하세요.
각 책마다 제목, 저자, 추천 이유(팀스파르타 문화와 연결), 연결된 핵심가치/그라운드룰 태그를 JSON으로 반환하세요.
```

### 유저 프롬프트
```
연차: {years_exp}
고민: {concern}
```
